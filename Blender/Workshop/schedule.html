Setting up

Changing theme

To change a theme, go to Blender User Preferences (or 'Ctrl + Alt + U' on Windows, or '⌘ Cmd + ,' on Mac). At tab Themes, on top left corner of the window, on Presets, click the box and choose the one you like! Elsyiun is my favorite. I recommend also changing the vertex size too! It's located on 3D View, on the same window. Scroll down to find Vertex Size and choose something higher than 3. I usually go with 7. To install a new theme, download and store the desired theme. Then at Blender User Preferences, in the Themes tab, you will see at the bottom of the window a Install Theme... button. Now you only have to locate the downloaded theme file and you are ready to go. Don't forget to save the changes of the configuration at Save User Settings button!

Learn the shorcuts

You will eventually remember the shortcuts, if you keep playing with it. Check the infographic, made by giudansky.com.

Installing addons

  • First things first: always save your addons into a folder you will easily find it later on. For example, I save them to D:\BlenderPackage\Addon\;
  • Don't forget to read thoroughly to know what you are getting from this addon, the recommendations etc;
  • There are cases which the author asks to unzip the file. In cases where there is no such step, don't unzip it. Just store it where you usually stores the addons;
  • Go to Blender User Preferences (or 'Ctrl + Alt + U' on Windows, or '⌘ Cmd + ,' on Mac);
  • At the Addons tab, go to Install from File...;
  • Find the newly downloaded .zip file then open it;
  • Check the box of the new addon from the list;
  • Don't forget to save the changes of the configuration at Save User Settings button!
