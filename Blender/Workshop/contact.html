Installing an add-on

Always save your addons into a folder you will easily find it later on. For example, I save them to D:\BlenderPackage\Addon\.

Let's use the quick preferences addon (author: Sean Olson (LiquidApe)) and measureit (author: Antonioya) as examples. The first is "QuickPrefs, Access from side panel to Lighting Presets and often changed preferences" and the last is "designed for displaying measures in the viewport, making the process of design objects with exact measures, easier."

Aside from the fact that they are very useful tools, we'll using them as examples because one is a single python script and another is a .zip file. We'll be covering on how to install these two types of addons.

As mentioned, we'll run into single python scripts, like the quick preferences addon, or to a .zip file like measureit.
